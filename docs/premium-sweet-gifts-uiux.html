Persistent Mini-Cart с анимацией добавления

Текущий ТЗ описывает mini-cart, но не уточняет поведение при добавлении товара. Необходима анимация «лёта» товара в корзину (fly-to-cart) и мгновенный slide-in drawer без перезагрузки страницы.

// Пример: анимация добавления в корзину const flyToCart = (element) => { const clone = element.cloneNode(true); clone.style.cssText = ` position: fixed; z-index: 9999; transition: all 0.6s cubic-bezier(0.25, 0.46, 0.45, 0.94); pointer-events: none; `; document.body.appendChild(clone); // анимация к иконке корзины };
📈 +12–18% Add-to-Cart rate по данным Baymard Institute
🔍
Высокий приоритет

Predictive Search с визуальными превью

Algolia настроен, но поисковый опыт можно усилить: показывать карточки товаров прямо в выпадающем результате (image + цена + кнопка) вместо plain-text списка.

/* Поисковый overlay с превью */ .search-overlay { display: grid; grid-template-columns: 1fr 340px; gap: 0; } .search-results { padding: 24px; } .search-preview { background: var(--cream); padding: 24px; border-left: 1px solid var(--cream-dark); /* live preview выбранного товара */ }
📈 +22% поисковых конверсий

Высокий приоритет

Прогрессивный Checkout (3 шага → 1 страница)

Разбейте checkout на визуальные шаги (адрес → доставка → оплата) без перезагрузки страницы. Accordion-стиль: каждый шаг раскрывается после подтверждения предыдущего.

// Checkout Step Machine const steps = [ { id: 'address', label: 'Адрес доставки', icon: '📍' }, { id: 'shipping', label: 'Способ доставки', icon: '📦' }, { id: 'payment', label: 'Оплата', icon: '💳' }, ]; // Каждый шаг — accordion panel, // completed шаги показывают summary
📈 −28% отказов на checkout

Wishlist с «Gift Hint» функцией

Delivery Date Picker прямо в корзине

Пользователь должен видеть ожидаемую дату доставки ДО перехода в checkout. Показывайте «Доставка к [дате]» с календарём выбора прямо в mini-cart и на странице товара.

// Расчёт даты на клиенте function getEarliestDelivery(country, method) { const days = DELIVERY_MAP[country][method]; const date = addBusinessDays(new Date(), days); return formatLocale(date, locale); // → "Вторник, 18 марта" }

Дизайн-система

CSS Custom Properties — полная палитра

Определите все токены на уровне :root. Это обеспечит консистентность и упростит тёмную тему или сезонные вариации.

/* Design Tokens — Premium Sweet Gifts */ :root { /* Brand Colors */ --color-gold: #C9A84C; --color-gold-light: #E8D5A3; --color-gold-dark: #8B6914; --color-cream: #FAF6EF; --color-ink: #1A1510; --color-error: #8B1F2C; /* Typography Scale */ --font-display: 'Cormorant Garamond', serif; --font-body: 'Jost', sans-serif; --text-xs: 0.75rem; /* 12px */ --text-sm: 0.875rem; /* 14px */ --text-base: 1rem; /* 16px */ --text-display: clamp(2rem, 5vw, 4rem); /* Spacing */ --space-xs: 4px; --space-sm: 8px; --space-md: 16px; --space-lg: 32px; --space-xl: 64px; --space-2xl: 128px; /* Motion */ --ease-luxury: cubic-bezier(0.25, 0.46, 0.45, 0.94); --duration-fast: 150ms; --duration-normal: 300ms; --duration-slow: 600ms; }

Микроанимации

Система micro-interactions через CSS

Единые анимации для всей платформы — кнопки, карточки, инпуты. Определите один раз, используйте везде.

/* Hover-lift для карточек товаров */ .product-card { transition: transform var(--duration-normal) var(--ease-luxury), box-shadow var(--duration-normal) var(--ease-luxury); } .product-card:hover { transform: translateY(-6px); box-shadow: 0 20px 60px rgba(26,21,16,0.12); } /* Button loading state */ .btn-primary:disabled { cursor: not-allowed; animation: shimmer 1.5s ease infinite; } @keyframes shimmer { 0%, 100% { opacity: 1; } 50% { opacity: 0.7; } }

Blurhash placeholder для изображений

При загрузке товаров показывайте размытый цветной placeholder (blurhash) вместо серого прямоугольника. Генерируется в Sanity CMS и передаётся как строка.

// Next.js Image с Blurhash import { decode } from 'blurhash'; <Image src={product.image.url} placeholder="blur" blurDataURL={product.image.blurHash} // Sanity возвращает lqip (base64) sizes="(max-width: 768px) 100vw, 50vw" alt={product.name} />
⚡ −40% perceived load time
📦
Высокий приоритет

Prefetch при hover на карточку

Next.js Link prefetch'ит страницу товара при наведении. Пользователь ещё не кликнул, а страница уже загружена — мгновенный переход.

// Prefetch при hover — Next.js import { useRouter } from 'next/navigation'; const router = useRouter(); <div onMouseEnter={() => router.prefetch(`/products/${slug}`) } > // карточка товара </div> // Или Link prefetch="intent" <Link href={href} prefetch={true}>
⚡ Переход в <100ms
🦴
Средний приоритет

Skeleton Screens для каталога

Вместо spinner — skeleton в форме карточки товара. Мозг воспринимает это как «что-то уже есть» и оценивает загрузку быстрее на 30%.

/* Skeleton анимация */ .skeleton { background: linear-gradient( 90deg, var(--cream-dark) 25%, var(--cream) 50%, var(--cream-dark) 75% ); background-size: 200% 100%; animation: skeleton-wave 1.5s ease infinite; } @keyframes skeleton-wave { 0% { background-position: 200% 0; } 100% { background-position: -200% 0; } }
⭐ +30% perceived performance

Высокий приоритет

Валюта и форматы по locale

Немцы пишут 1.234,56 € — а не €1,234.56. Интл форматирование чисел, дат и адресов критично для доверия. Используйте Intl.NumberFormat повсеместно.

// Форматирование цен по locale const formatPrice = (amount, locale, currency) => new Intl.NumberFormat(locale, { style: 'currency', currency, minimumFractionDigits: 2, }).format(amount); // DE: 1.234,56 € // FR: 1 234,56 € // EN: €1,234.56 // RU: 1 234,56 €
⭐ Доверие и профессионализм
📅
Низкий приоритет

Локализованные праздники в «поводах»

Фильтр «повод для подарка» должен учитывать местные праздники: немцам показывать Oktoberfest, испанцам — Día de la Madre (отличается от EN), итальянцам — Festa della Repubblica.

// Locale-specific occasions const occasions = { en: ['Christmas', 'Valentine's Day', ...], de: ['Weihnachten', 'Oktoberfest', ...], es: ['Navidad', 'Día de la Madre', ...], it: ['Natale', 'Festa della Repubblica', ...], fr: ['Noël', 'Fête des Mères', ...], ru: ['Новый год', '8 марта', ...], };
